dicomParser Examples

These examples are meant to demonstrate various aspects of the dicomParser library in an interactive way. They are written such that they can be opened directly in your web browser from github or your local repository. Use the "view source" capabilities built into your browser to see how each example works.

  • Getting Started Walkthrough of how to use the library
  • AMD Example showing how to use the library as an AMD module in Require.js
  • DICOM Dump Drag and drop a DICOM file onto the web page to see a dump of all elements
  • DICOM Summary Drag and drop a DICOM file onto the web page to see a summary of its contents
  • DICOM Dump with data dictionary Drag and drop a DICOM file onto the web page to see a summary of its contents including element names based on a data dictionary
  • WADO DICOM Dump Enter a URL to a WADO image and see its contents
  • Node.js Example Example of using dicomParser with Node.js
  • Meteor Example Example of using dicomParser with Meteor
  • ExplicitDataSetToJS Example of using the utility function explicitDataSetToJS to convert a dataSet into a JavaScript object
  • ExplicitDataSetToJSON Example of using the utility function explicitDataSetToJS to convert a dataSet into JSON
