(static) bbox(center, halfDimensions) → {Object|boolean}

Generate a bouding box object.
Parameters:
Name Type Description
center THREE.Vector3 Center of the box.
halfDimensions THREE.Vector3 Half Dimensions of the box.
Source:
  • core/core.utils.js, line 39
Returns:
  • The bounding box object. {Object.min} is a {THREE.Vector3} containing the min bounds. {Object.max} is a {THREE.Vector3} containing the max bounds.
    Type
    Object
  • False input NOT valid.
    Type
    boolean
Example
// Returns
//{ min: { x : 0, y : 0,  z : 0 },
//  max: { x : 2, y : 4,  z : 6 }
//}
VJS.Core.Utils.bbox( new THREE.Vector3(1, 2, 3), new THREE.Vector3(1, 2, 3));

//Returns false
VJS.Core.Utils.bbox(new THREE.Vector3(), new THREE.Matrix4());
